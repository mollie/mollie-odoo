How to configure

Step 1 Download the module and place the module in apps folder.

Step 2 Install this module in your Odoo instance.

Step 3 Open Point of Sale > Configuration > Settings and find Payment Terminals section

Step 4 Fill the API key. (You will find these keys from your Mollie dashboard.)

Step 5 After saving api keys, Go to Point of Sale > Configuration > Mollie Pos Terminal and click on sync terminal.

Step 6 Now go to pos Payment Methods and select mollie as payment terminal.

DONE That's it you are ready to accept payments.
