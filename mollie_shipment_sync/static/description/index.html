Sync shipment data from odoo to mollie.

Make sure you are using Mollie payment module >= 12.0.1.5

How to configure

Step 1 Download the module and place the module in apps folder and install the module.

How to Use it

Step 1 After installation, you will see the Sync Shipment button on order (if it is paid with mollie)

Step 2 Clicking on Sync Shipment button will sync shipment data with mollie.

You can enable, automatic shipment sync from the settings of mollie payment.
